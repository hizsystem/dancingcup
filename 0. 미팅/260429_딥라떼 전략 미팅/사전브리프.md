## 2. 마케팅팀 노션 요점

원본: 노션 페이지 "댄싱컵 딥라떼 라인업 기획 방향" (7장 구성)

### A. 브랜드 목표 (01)
- 딥라떼 = "댄싱컵 = 라떼 맛집" 포지셔닝 앵커
- **3단계 인지 빌드**: ① 브랜드 인지 → ② 카테고리 연상(라떼 맛집) → ③ 시그니처 인지(딥라떼)

### B. 메인 카피 후보 (02)
- 라떼하우스 댄싱컵 / 라떼마스터 댄싱컵
- **깊이가 다른 라떼, 댄싱컵 / 더 깊게 즐기는 라떼, 댄싱컵 / 라떼의 깊이, 댄싱컵**
- 키워드 = "깊이"

### C. 라인업 (03) — 마케팅팀 안
- **커피 라인 ONLY** (논커피 제외 — '라떼 맛집' 포지셔닝 희석 우려)
- **기존 베이스 + 변주 4종 안에서 3종 선택**
  - ① 오리지널 딥라떼 (기준점)
  - ② 아몬드/너티 딥라떼 — 데니스·마일스톤·매머드·텐퍼센트 더티땅콩·우지커피 피넛크림 벤치
  - ③ 메이플 딥라떼 — 팀홀튼·영앤도터스 벤치
  - ④ 쏠티드 딥라떼 — 텐퍼센트 솔티드밀크카라멜·유동커피 소금우유라떼 벤치
- **가격 = 기존 +200~300원** (전략적으로 비싸야 있어보인다는 인식)

### D. PHASE 운영 (04)
- **PHASE 1**: 오리지널 + 변주 2종 = **총 3종** 런칭 (단순 라인업 = 인지도 확보)
- **PHASE 2**: 시즈널 확장 (스파이시 — 시나몬·카다멈 · 가을·겨울 / 시트러스 — 레몬·오렌지 · 봄·여름)
- "메뉴 반응 보며 넣었다 빼는 것은 마케팅 힘이 안 실림" 입장

### E. 비주얼 방향 (05)
- PHASE 1: 라떼 깊이감
- PHASE 2: **세계관** 시각화 / 원물 시각화 / **마블링 패턴** (컵홀더·부자재 활용 가능)

### F. 마케팅 키워드 (06) — ★ HIZ 의견서와 동일 노선
- **'부산딥라떼' 잡으면 좋겠다** (노션 명시!)
  - "부산딥라떼가 떴다", "부산 가면 꼭 먹어야 하는 부산딥라떼"
- **텐퍼센트 꾸덕라떼** = 소비자 언어로 만들어진 키워드 사례 (HIZ 킥오프 덱과 동일)
- "댄싱커피"는 직관적이지 않아서 안 가지고 가도 됨
- "가심비"는 교과서적이라 매력 X
- **"스타벅스 절반 가격"** = 가격저항성 큰 사람한테 메리트, *소재 테스트용*

### G. 숏폼 플레이 (07)
- PHASE 1: 릴스/쇼츠/틱톡 채널 콘텐츠 제작
- PHASE 2: 콘텐츠 유형 5가지
  - **ASMR형** (제조 소리·매장 ambience) → 신뢰·안정감
  - **일상 Vlog형** (오픈 루틴·손님 에피소드·마감 후 리뷰) → 친근감·단골
  - **꿀조합형** (커스텀 레시피·계절 한정 추천) → 정보 공유·방문 유도
  - **비하인드형** (재료 준비·신메뉴 테스트·원두 입고) → 전문성·신뢰
  - **신메뉴 홍보형** → 신메뉴 노출
  - 본사 가이드: 해시태그·첫 3초 노출·경쟁사 직접 언급 금지·유니폼/위생
- PHASE 3: **앰배서더 = 구글폼 공개 모집** (점주 자기 지원 → 선발)
- PHASE 4: 인센티브
  - **현금성**: 월 활동비 20~30만원 (제작자 직접 지급) + 조회수 1만·5만 보너스 + 월간 1등 상금
  - **점주/매장**: 월정 물류비·원두 지원 + 촬영 전용 원부재료 본사 발송
  - **커리어**: 본사 인증 앰배서더 임명장

## 3. 양 팀 합의·미합의

### A. 이미 합의된 것 (행동 가능)

- 딥라떼 = 댄싱컵 "라떼 맛집" 포지셔닝 앵커
- 출시 시점 = **7~8월 여름 시즌** (아이스 특화)
- 가격 = 오리지널 3,300 / 변주 3천원 중후반대
- 시즈널 확장 로드맵 — 가을·겨울 라인업 추가로 본사 아이덴티티 정착
- **"부산딥라떼" 키워드 활용** (마케팅팀 노션 06 명시 + HIZ 의견서)
- 매장 = 로컬 마케팅 채널 (가맹점 SNS·숏폼 활성화)
- 비주얼 차별화 자산 — 마케팅팀 마블링 패턴 + 운영팀 "딥 베이스 숙성 중" 스티커

### B. 오늘 결정해야 할 것 — 미합의 4건

| # | 항목 | 운영팀 안 | 마케팅팀 안 | 결정축 |
|---|------|----------|------------|--------|
| ① | **라인업 수·카테고리** | 4종 (오마주, 2/4 중 1택 시 3종 가능) — 너티봉봉(논커피) 포함 | **3종** (오리지널 + 변주 2) — **커피만, 논커피 제외** | 3 vs 4 / 논커피 |
| ② | **메뉴 방향** | 친숙 **오마주** (스벅·베라·롯데·차백도) | 베이스 **변주** (고소·달콤·쏠티) | 오마주 vs 변주 (혼합 가능) |
| ③ | **앰배서더 선발** | 진해신항·거제장평 **우선 임명** + 누적 조회수 챌린지 | 구글폼 **공개 모집** → 선발 | 임명 vs 공모 (병행 가능) |
| ④ | **인센티브 단가** | 원두 1봉/회 (월 1~3만 환산) · 전체 월 172만 | **월 활동비 20~30만 + 보너스 + 상금** | 현물 vs 현금 (예산 격차 약 10배) |

### C. 미팅에서 결정 가능한 합의안 후보

- **①**: 운영팀 4종 안에서 너티봉봉 빼고 3종 (스위트밀크·BG·오리지널) — 커피만 + 오마주 골격
  - 또는 마케팅팀 3종 골격(오리지널+고소·달콤 또는 쏠티) + 메뉴명에 운영팀 오마주 토핑 결합
- **②**: *오마주 골격 + 부산 변주 어휘*의 하이브리드 — 양 팀 양립
- **③**: *진해·거제 1차 임명 + 2차 공모* 병행 → 부산 앵글에 자연 정합 (HIZ 의견서 슬라이드 07 권고)
- **④**: 위드런 내부 예산 결정 사안. HIZ는 입장 없음. 단 **결정된 단가가 부산 앵글 콘텐츠 KPI에 연동되면** 채널 운용 일관성 확보 가능

## 4. HIZ 의견서 (v2) 톤

### 단 한 가지를 강하게 미는 의견서
- **"부산에서 온 라떼 맛집" 우산 채택 — 강도 B**
- 메뉴명·가격·패키지·라인업 구성은 **불변**. 시그니처 카피·해시태그·매장 POP·콘텐츠 어휘만 부산화
- 운영 전환 비용 ≈ 0. 여름 라인업이 3종이든 4종이든, 시즈널 확장 어떤 안과도 호환
- **양 팀이 이미 같은 자리를 보고 있다** — 운영팀 자산(부산·경남 130호점) + 마케팅팀 키워드("부산딥라떼")

### 미팅에서 HIZ가 강하게 미는 카드 = 1개뿐
- 의견서 슬라이드 09: ① 부산 앵글 우산 채택 / ② 핵심 키워드 3종 합의 / ③ 여름 D-7 점화 일정 가동
- ④ 미합의(라인업 수·메뉴 방향·앰배서더·인센티브)는 양 팀 합의 사안. **HIZ는 정렬축만 깔고 자세 빠짐**

### 의견서가 운영팀·마케팅팀 자료를 녹이는 방식

### A. 미팅 정치 안전선
- HIZ는 ④ 미합의 4건 중 어느 한 쪽 편 들지 않는다 (특히 인센티브 단가 — 위드런 예산 결정)
- 우리가 가져가는 건 *우산 1건* + *키워드 3종* + *D-7 일정* 만. 양 팀 합의에 끼어들지 않는다

### B. 미팅에서 우리가 들어야 하는 것
- 운영팀: 4종 vs 3종 / 너티봉봉 운명 / D-Day 정확한 날짜 (7월 또는 8월)
- 마케팅팀: 메인 카피 후보 ("깊이" 키워드)와 부산 앵글 우산이 어떻게 결합될지 / 비주얼 마블링 시안 진척도 / 앰배서더 공모 일정
- 양 팀 공통: 인센티브 단가 결정 (HIZ 콘텐츠 KPI 연동 가능 여부)

### C. 미팅 후 HIZ 즉시 작업
- **합의되면**:
  1. HIZ 결정로그 (`_PM/결정로그.md`) 등재
  2. 마스터시트 키워드 데이터 탭 갱신 (#부산딥라떼 등)
  3. 인스타·숏폼·X 채널 브리프에 부산 어휘 일괄 반영
  4. 매장 POP / 쇼룸 스티커 카피 1차 시안 5/10까지 제출
  5. D-Day 확정 후 D-7 타임라인 가동
- **미합의되면**:
  1. 미팅 녹취 기록 → `0. 미팅/260429_딥라떼 전략 미팅/`
  2. HIZ 결정로그 Pending 큐에 보존
