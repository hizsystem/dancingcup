## 3. 활동 구조

### 3-1. 인원·기간
- **5집 (5명)** — 4/29 결정. 매장 분포 = 부산 3 / 경남 1 / 기타 1 권장
- **6개월 활동** (예: 7월 D-Day 직전 6월 모집 → 7월~12월 활동)
- 종료 후 평가·갱신 또는 신규 모집

### 3-2. 월별 콘텐츠 의무
| 항목 | 횟수 | 형식 |
|---|---|---|
| **신메뉴·시즈널 콘텐츠** (가이드 통일) | 월 1회 | HIZ 가이드 + 본인 스타일 |
| **자유 콘텐츠** | 월 3회 | 본인 자유 (단 댄싱컵 매장·메뉴 노출) |
| 합계 | **월 4개** | 5집 × 4개 = **월 20개** |

### 3-3. 콘텐츠 가이드
- **통일 가이드 = 신메뉴·시즈널만** (예: 7월 D-Day 솔티드 메인 컷·후크 통일)
- **자유 콘텐츠 = 자유도 ↑** — 4/29 미팅 결정 (미러링 + 자유도 병행)
- 금지 사항만 명시: 경쟁 브랜드 직접 디스 / 가심비·가성비 어휘 / 정치·종교

## 4. 인센티브 — 동력의 핵심 (허들 ↓ × 인센티브 ↑)

### 4-1. 기본 활동비
- **월 정액 30만원/명** × 5명 = 월 150만원
- 콘텐츠 4개 의무 충족 시 지급

### 4-2. 조회수 보너스 (4/29 미팅 합의)
| 1개 콘텐츠 조회수 | 보너스 |
|---|---|
| 1만 회 돌파 | +5만원 |
| 5만 회 돌파 | +20만원 |
| 10만 회 돌파 | +50만원 |
| 30만 회+ | +100만원 + 다음 시즌 우선 갱신 |

> 단일 콘텐츠당 보너스. 누적 합산. 인플루언서 1건(보통 50만원~) 대비 합리적이고, 알바생 입장에선 큰 동력.

### 4-3. 매장 인센티브
- 앰버서더 매장의 월 매출 +α 시 매장 단위 보너스 (점주에게 추가 동력)
- 분기 1회 베스트 앰버서더 시상 (매장 회식 비용 등)

## 5. 콘텐츠 검수·재배포 룰

### 5-1. 검수
- **사전 검수 = 신메뉴·시즈널 콘텐츠만** (HIZ가 가이드 핵심 컷 확인)
- **자유 콘텐츠 = 사후 검수** (업로드 후 HIZ 모니터링)
- 검수 시간 ≤ 24시간 — 빠른 피드백이 동력 유지의 핵심

- 앰버서더 콘텐츠를 HIZ가 공식 인스타·릴스에 재배포

- 매장 130개 미러링 가이드에 자동 포함 (자발 미러링 유도)

### 5-3. UGC 해시태그 운영

## 7. 예산 시뮬레이션

### 7-1. 월별 비용 (5집 기준)
| 항목 | 금액 |
|---|---|
| 활동비 | 30만 × 5 = 150만 |
| 조회수 보너스 (예상치) | 50-100만 |
| 매장 인센티브 (분기) | 분기당 50만 (월 평균 17만) |
| **월 합계** | **약 220-270만** |

### 7-2. 6개월 누적 (7-12월)
- 월 250만 × 6개월 = **약 1,500만**
- 댄싱컵 리테이너 500만/월 × 6 = 3,000만 중 50% 비중
- 나머지 50% = 콘텐츠 제작·인플루언서·메타 광고·운영

> **인플루언서 1건 비교**: 마이크로 인플루언서 1건 = 50-100만 / 콘텐츠 1개. 앰버서더 5집 = 월 20개 = **콘텐츠 단가 12-13만원** — 인플루언서 대비 1/5~1/8 효율.
